Código para alterar nome da branch master para main

git branch -m master main

Configurando o git: informações credenciais

git config --local user.name "Seu nome aqui"

git config --local user.email "seu@email.aqui"

Comando para ver o status dos arquivos a serem salvos

git status

Comando para adicionar arquivos para serem salvos

git add nomeDoArquivo git add *

Comando para comitar

git commit -m "mensagem do commit"

Comando para ver histórico de commits

git log

Comando para verificar email

git config user.email

Comando para verificar nome de usuario

git config user.name

Comando que mostra o historico de commits com cada commit em uma linha

git commit --oneline

git remote

git remote add nomeDoArquivo

Comando para criar pastar no terminal windows

mkdir nomeDaPasta

git clone caminho/url do repositorio

Comando para empurrar os dados para outro repositorio

git push localDeOrigem localDeChegada exemplo: git push origin main

git pull nomeDoRepositorioRemoto nomeDaBranchDoRepositorioLocal

Comando para renomear repositorios remotos

git remote rename nomeAtual nomeDesejado

Comando para entrar em um pasta

cd nomeDaPasta

cd ../

git init --bare

git congif -v

Comando para verificar a branch em que esta

git branch

Mudar o nome da branch

git branch nomeDaBranch

Comando para briar um branch e ja entrar nele

Juntando Branches

git merge nomeDaBranch com a qual quer dar merge

deletando Branches

git branch -d "nome da branch"

Comando para juntar duas branches

Comando para juntar duas branches sem precisar fazer outro commit depois para confirmar as mudanças

Comando para voltar alterações que ainda nao foram salvas

Remover arquivos adicionados para serem commitados

(WINDOWS/LINUX MINT) git reset HEAD nomeDoArquivo (LINUX MINT) git restore --staged nomeDoArquivo

Comando para criar um comit para desfazer um comit

Salvando alteraçoes no stash para usar depois

git stash

Lista de stashes

git stash list

Juntando mudanças feitas com o stash

git stash apply numeroDaStash

apagando a stash da lista

git stash drop numeroDaStash

trazendo alteraçoes da stash e apagando ao mesmo tempo

git stash pop numeroDaStash

Navegando entre commit - mover para o commit desejado e depois criar um novo Branch, do contrario as mudanças feitas não poderão ser salvas

git checkout hashDoCommit

Ver alteraçoes feitas entre comits

git diff hashDeUmComite..hashDeOutroComit

Ver alteraçoes feitas e que nao foram adicionadas

git diff

git tag

git tag nomeDoRepositorioRemoto nomeDaTag
